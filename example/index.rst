=================================
Material Design Theme For Sphinx
=================================

| Sphinx Material Design Theme is HTML theme for `Sphinx <http://sphinx.pocoo.org/>`_.
| This theme has been using `Google Material Design Lite <https://getmdl.io/>`_.

Quick Start
===========

Install the latest version of sphinx_symbiflow_theme with :code:`pip`.

.. code-block:: bat

   pip install git+http://github.com/SymbiFlow/sphinx_materialdesign_theme.git@master#egg=sphinx_symbiflow_theme

Add the following line to :code:`conf.py`.

.. code-block:: python

   html_theme = 'sphinx_symbiflow_theme'
